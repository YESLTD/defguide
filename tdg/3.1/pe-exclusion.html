%*.exclusion; Parameter Entities

Name

%*.exclusion; Parameter Entities -- Control SGML exclusions

Synopsis

Used to control the exclusion declarations on elements with exclusions.

The %*.exclusion; parameter entities parameterize element exclusions. You can selectively include or remove elements from the exclusion by changing the appropriate parameter entity.
