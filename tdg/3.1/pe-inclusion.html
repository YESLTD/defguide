%*.inclusion; Parameter Entities

Name

%*.inclusion; Parameter Entities -- Control SGML inclusions

Synopsis

Used to control the inclusion declarations on elements with inclusions.

The %*.inclusion; parameter entities parameterize element inclusions. You can selectively include or remove elements from the inclusion by changing the appropriate parameter entity. For example, if you remove BeginPage from the %ubiq.inclusion; parameter entity, it is no longer included everywhere in DocBook.
