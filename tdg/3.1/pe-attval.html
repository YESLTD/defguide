%*.attval; Parameter Entities

Name

%*.attval; Parameter Entities -- Define attribute values

Synopsis

These parameter entities define attribute values used on selected attributes.

%no.attval;

Defines the value for "no".

Parameter Entity

0

%no.attval; contributes to the declaration of the following elements:

FigureInformalFigure

%yesorno.attvals;

Defines the attribute values for "boolean" attributes.

Parameter Entity

NUMBER

%yesorno.attvals; contributes to the declaration of the following elements:

FigureGraphicImageData
InformalFigureInformalTableInlineGraphic
VideoData
